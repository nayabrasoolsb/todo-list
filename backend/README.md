# Todo List

This is a **backend Todo List project** built using **NestJS**, **Prisma ORM (v7)**, **TypeScript**, and **PostgreSQL**.


---

##  Install Dependencies

```bash
npm install
```

---

##  Prisma Commands

```bash
# Generate Prisma Client (run after schema changes)
npx prisma generate

# Push Prisma schema to database (development only)
npx prisma db push

# Open Prisma Studio (database GUI)
npx prisma studio
```

---

## To run the project

```bash
# Run in development mode
npm run start:dev
```
